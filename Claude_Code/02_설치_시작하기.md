# 02. 설치 & 시작하기

> Claude Code를 설치하고 첫 번째 작업을 실행하는 방법을 단계별로 설명합니다.

---

## 사전 준비

### 필수 요건

| 항목 | 최소 버전 | 확인 방법 |
|------|-----------|-----------|
| Node.js | 18 이상 | `node --version` |
| npm | 8 이상 | `npm --version` |
| Anthropic API 키 | — | console.anthropic.com에서 발급 |

### API 키 발급

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 계정 생성 또는 로그인
3. **API Keys** 메뉴 → **Create Key**
4. 발급된 키를 안전한 곳에 저장 (한 번만 표시됨)

> API 키는 `sk-ant-...` 형태입니다. 절대 코드에 직접 넣거나 Git에 올리지 마세요.

---

## 설치

```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:

```bash
claude --version
```

---

## API 키 설정

Claude Code를 처음 실행하면 API 키를 묻습니다. 다음 세 가지 방법 중 하나로 설정하세요.

### 방법 1: 환경변수 (권장)

```bash
# macOS / Linux
export ANTHROPIC_API_KEY="sk-ant-..."

# Windows PowerShell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

영구 적용하려면 `~/.zshrc`, `~/.bashrc`, 또는 `~/.profile`에 추가하세요.

### 방법 2: 실행 시 직접 입력

`claude` 명령어를 실행하면 API 키를 입력하라는 프롬프트가 나타납니다. 입력하면 로컬에 저장됩니다.

### 방법 3: .env 파일 (프로젝트별)

프로젝트 루트에 `.env` 파일을 만들고 `.gitignore`에 추가하세요.

```
ANTHROPIC_API_KEY=sk-ant-...
```

---

## 첫 실행

프로젝트 디렉토리로 이동한 뒤 실행합니다.

```bash
cd my-project
claude
```

처음 실행 시 약관 동의 화면이 나타납니다. 동의하면 대화 프롬프트가 열립니다.

```
✓ Connected to Claude (claude-sonnet-4-6)
> 
```

이제 자연어로 무엇이든 요청할 수 있습니다.

```
> 이 프로젝트 구조를 설명해줘
> src/api/users.ts 파일에서 에러 처리가 누락된 부분을 찾아줘
> README.md를 업데이트해줘
```

---

## 프로젝트 초기화

새 프로젝트에서 Claude Code를 처음 사용할 때는 `/init`으로 초기화합니다.

```bash
/init
```

Claude가 프로젝트를 분석해서 `CLAUDE.md` 파일을 자동으로 생성합니다. 이 파일은 Claude가 프로젝트를 이해하는 기준 문서입니다.

---

## 실행 방법 옵션

```bash
claude                     # 대화 모드로 시작
claude "버그 찾아줘"        # 한 줄 명령으로 실행
claude --continue          # 마지막 세션 이어서 시작
claude --resume            # 특정 세션 선택해서 재개
claude --model opus        # 특정 모델로 시작
```

## 자주 발생하는 설치 문제

### "command not found: claude"

npm global 경로가 PATH에 없는 경우입니다.

```bash
npm bin -g   # global bin 경로 확인
# 출력된 경로를 PATH에 추가
```

### "API key not found"

환경변수가 설정되지 않았거나 현재 셸에 반영되지 않은 경우입니다.

```bash
echo $ANTHROPIC_API_KEY   # 값 확인
source ~/.zshrc            # 설정 파일 다시 로드
```

### "Permission denied"

```bash
npm install -g @anthropic-ai/claude-code --prefix ~/.local
```
