## Claude Code가 뭔가요? (30초 요약)

**Claude Code**는 Anthropic이 만든 AI 코딩 어시스턴트로, 터미널에서 직접 실행합니다.

일반 챗봇과 다른 점은, 단순히 대화만 하는 게 아니라 **실제로 코드를 파악하고 코드를 작성하며 명령어를 실행하고 Git을 다루는** 등 개발자가 하는 작업을 직접 수행할 수 있다는 것입니다.

```
개발자  →  터미널에서 Claude Code 실행
        →  "이 버그 고쳐줘" 라고 말함
        →  Claude가 파일을 읽고, 코드를 수정하고, 테스트까지 실행
        →  결과 확인 후 "좋아" 또는 "다시 해줘"
```
